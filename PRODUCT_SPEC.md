# 출퇴근길 익명 채팅 - 기획 명세서

## 📋 프로젝트 개요

### 서비스명
**출퇴근길 익명 채팅**

### 컨셉
서울 지하철 호선별로 나뉜 익명 채팅방에서 같은 호선을 이용하는 통근러들이 자유롭게 소통하는 플랫폼

### 핵심 가치
- 🚇 **호선 기반 커뮤니티**: 같은 노선을 이용하는 사람들끼리의 공감대
- 🎭 **완전한 익명성**: 부담 없는 자유로운 소통
- 💬 **실시간 채팅**: 게시판이 아닌 메신저 형태의 즉각적인 대화
- 🔒 **안전한 환경**: 매일 자동 삭제되는 메시지로 흔적 없는 대화

---

## 🎯 타겟 사용자

### 주 타겟
- 서울 지하철 통근/통학자
- 20-40대 직장인
- 출퇴근 시간 여유 있는 사용자

### 사용 시나리오
1. **출근길 지하철 내**: "오늘 회사 가기 싫다"
2. **퇴근길 정류장**: "오늘 야근 언제 끝나려나"
3. **점심시간**: "이번 역 근처 맛집 추천해주세요"
4. **집에서**: "내일 출근길 날씨 어때요?"

## ✨ 주요 기능

### 1. 로그인 시스템

#### 로그인 방식 (2가지)

**1) 익명으로 시작하기** ⭐ 추천
- **특징**: 클릭 한 번으로 즉시 입장
- **ID**: 자동 생성 (예: `anon_a1b2c3d4`)
- **닉네임**: 랜덤 생성 (예: "활기찬 통근러123", "즐거운 출퇴근러456")
- **제약**: 본인 메시지 삭제 불가
- **장점**: 빠른 진입, 완전한 익명성

**2) 카카오로 시작하기**
- **특징**: 카카오 계정 연동
- **닉네임**: 카카오 프로필 닉네임
- **권한**: 본인 메시지 삭제 가능
- **장점**: 안정적인 계정 관리

#### 로그인 필수
- 모든 서비스 이용 전 로그인 필수
- 비로그인 상태에서는 호선 목록만 볼 수 있음
- 호선 클릭 시 자동으로 로그인 페이지로 이동

---

### 2. 호선 선택

#### 화면 구성
```
┌─────────────────────────────────┐
│  출퇴근길 익명 채팅         [로그인] │
├─────────────────────────────────┤
│                                 │
│   [1호선] [2호선] [3호선] [4호선]  │
│   [5호선] [6호선] [7호선] [8호선]  │
│   [9호선]                       │
│                                 │
│   익명으로 가볍게 출퇴근길         │
│   이야기를 나눠보세요.            │
│                                 │
└─────────────────────────────────┘
```

#### 호선별 색상
- 1호선: 남색
- 2호선: 초록색
- 3호선: 주황색
- 4호선: 하늘색
- 5호선: 보라색
- 6호선: 갈색
- 7호선: 올리브색
- 8호선: 분홍색
- 9호선: 금색

#### 인터랙션
- 호선 버튼에 마우스 오버: 확대 효과
- 클릭: 해당 호선 채팅방 입장
- 비로그인 상태: 로그인 페이지로 리다이렉트

---

### 3. 채팅방

#### 화면 구성
```
┌─────────────────────────────────┐
│ ← [1호선 채팅]          [로그아웃] │
├─────────────────────────────────┤
│                                 │
│  ┌─────────────────────────┐   │
│  │ 활기찬 통근러123          │   │
│  │ 오늘 회사 가기 싫다 ㅠㅠ    │   │
│  │                    09:15 │   │
│  └─────────────────────────┘   │
│                                 │
│  ┌─────────────────────────┐   │
│  │ 즐거운 출퇴근러456         │   │
│  │ 저도요 ㅋㅋㅋ              │   │
│  │                    09:16 │   │
│  └─────────────────────────┘   │
│                                 │
├─────────────────────────────────┤
│ [메시지 입력...]          [전송] │
└─────────────────────────────────┘
```

#### 메시지 형식
- **닉네임**: 상단에 표시
- **내용**: 말풍선 형태
- **시간**: 우측 하단 (09:15 형식)
- **삭제 버튼**: 카카오 로그인 사용자만, 본인 메시지에만 표시

#### 채팅 특징
- **실시간 업데이트**: 5초마다 새 메시지 확인
- **자동 스크롤**: 새 메시지 도착 시 자동으로 최하단 이동
- **최근 100개**: 최근 100개 메시지만 표시
- **매일 삭제**: 매일 자정 모든 메시지 자동 삭제

#### 메시지 입력
- **텍스트만 지원**: 이미지/파일 첨부 불가
- **글자 수 제한**: 1-1000자
- **전송 방법**:
  - Enter: 메시지 전송
  - Shift + Enter: 줄바꿈
  - 전송 버튼 클릭

## 🔐 보안 및 개인정보

### 개인정보 처리
- **익명 사용자**: 개인정보 수집 없음
- **카카오 로그인**: 카카오 ID, 닉네임만 저장
- **메시지**: 작성자 정보 저장 (익명은 NULL)

### 데이터 보관
- **메시지**: 매일 자정 자동 삭제
- **사용자 정보**: 회원탈퇴 시 즉시 삭제 (현재 미구현)
- **로그**: 최소한의 접근 로그만 보관

### 보안 조치
- **HTTPS**: 모든 통신 암호화
- **JWT**: 안전한 인증 토큰
- **Rate Limiting**: API 남용 방지 (15분당 100회)
- **XSS 방지**: 입력값 검증 및 이스케이프
- **SQL Injection 방지**: Prepared Statement 사용

## 🎯 차별화 포인트

### 기존 서비스와의 차이

#### vs. 에브리타임
- **차이점**: 호선 기반 (학교 기반 X)
- **장점**: 더 넓은 대상, 출퇴근 특화

#### vs. 블라인드
- **차이점**: 완전 익명 (회사 인증 X)
- **장점**: 진입 장벽 낮음, 가벼운 대화

#### vs. 카카오톡 오픈채팅
- **차이점**: 자동 삭제 (영구 보관 X)
- **장점**: 흔적 없는 대화, 부담 없음

### 핵심 경쟁력
1. **호선 기반 세분화**: 공감대 형성 용이
2. **익명성 + 간편함**: 클릭 한 번으로 참여
3. **자동 삭제**: 책임감 낮춰 활발한 참여 유도
4. **출퇴근 특화**: 시간대별 트래픽 예측 가능

## 📱 사용자 여정 (User Journey)

### 첫 방문자
```
1. 서비스 발견 (SNS, 검색)
   ↓
2. 홈페이지 접속 → 호선 목록 확인
   ↓
3. 호선 클릭 → 로그인 페이지
   ↓
4. "익명으로 시작하기" 클릭 (3초 소요)
   ↓
5. 채팅방 입장 → 최근 대화 확인
   ↓
6. 메시지 작성 → 실시간 반영
   ↓
7. 다른 사용자 반응 확인 → 만족
```

### 재방문자
```
1. 북마크/즐겨찾기로 접속
   ↓
2. 자동 로그인 (localStorage)
   ↓
3. 호선 선택 → 즉시 채팅방 입장
   ↓
4. 활발한 참여
```

## 🎭 페르소나

### 페르소나 1: "빠른 익명이"
- **연령**: 28세 직장인
- **통근**: 2호선 (강남 → 을지로)
- **니즈**: 출퇴근 시간 심심함 해소
- **행동**: 매일 아침 출근길 5분, 퇴근길 10분 참여
- **선호**: 익명 로그인, 가벼운 대화

### 페르소나 2: "소통하는 카카오"
- **연령**: 35세 직장인
- **통근**: 7호선 (노원 → 청담)
- **니즈**: 같은 노선 이용자들과 정보 공유
- **행동**: 주 3-4회 참여, 맛집/교통 정보 공유
- **선호**: 카카오 로그인, 본인 메시지 관리

### 페르소나 3: "정보 찾는 학생이"
- **연령**: 22세 대학생
- **통근**: 4호선 (사당 → 혜화)
- **니즈**: 출퇴근 시간대 교통 정보, 맛집 추천
- **행동**: 필요할 때만 검색 후 방문
- **선호**: 익명 로그인, 빠른 정보 획득
