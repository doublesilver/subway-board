## 주요 화면 구성

### 1. 메인 화면 (호선 선택)

**구성 요소**:
- 헤더: 서비스명 + 설명
- 호선 카드 리스트 (1~9호선)
- 각 호선 고유 색상 배경
- 현재 활동 인원 표시

### 2. 채팅방 화면

**구성 요소**:
- 헤더: 뒤로가기 + 호선명
- 메시지 영역 (스크롤 가능)
- 입력창 (하단 고정)

**말풍선 디자인**:
- **내 메시지**: 검정 배경, 우측 정렬
- **상대 메시지**: 연한 회색 배경, 좌측 정렬
- **시스템 메시지**: 중앙 정렬, 텍스트만

**입력창**:
- 자동 높이 조절 (최대 120px)
- 전송 버튼: 44px 원형
- Safe Area Bottom 대응

## UX 원칙

### 1. 한 손 조작 최적화
- 중요 버튼은 하단 1/3 영역에
- 스와이프 제스처 지원 (답장)
- 큰 터치 영역 (최소 44px)

### 2. 모바일 퍼스트
- 세로 모드 최적화
- 뷰포트: 375px ~ 430px 대응

### 3. 성능
- 부드러운 스크롤
- 즉각적인 피드백 (active 상태)
- 로딩 최소화

### 4. 익명성
- 로그인 불필요
- 자동 생성 닉네임
- 매일 자정 리셋으로 임시성 강조
