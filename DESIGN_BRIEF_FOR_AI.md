# UI/UX 디자인 리퀘스트 프롬프트

## 📱 프로젝트 개요

**서비스명**: 출퇴근길 익명 채팅 (서울 지하철 호선별 커뮤니티)

**핵심 컨셉**:
- 출퇴근 시간에 같은 지하철 노선을 이용하는 사람들끼리 익명으로 가볍게 소통
- 매일 오전 9시 자동 리셋으로 부담 없는 일회성 대화
- 모바일 웹 전용, PWA 지원

**타겟 유저**:
- 20-30대 직장인
- 출퇴근 시간(왕복 1-2시간) 스마트폰 사용자
- 한 손으로 지하철 손잡이 잡고 엄지로만 조작
- 짧은 세션(5-10분), 하루 2회 정도 사용

## 🎯 디자인 요구사항

### 1. 전체 톤앤매너

**감성 키워드**:
- ✅ **모던 미니멀리즘**: 불필요한 장식 없이 깔끔하고 세련됨
- ✅ **친근하지만 가볍게**: 과도하게 귀엽거나 무겁지 않은, 절제된 친근함
- ✅ **도시적 감성**: 서울 지하철, 출퇴근, 도심 라이프스타일
- ✅ **일시적 공간**: 매일 리셋되는 임시 대화방 느낌

**지향하는 레퍼런스**:
- Threads (Meta) - 심플하고 대화 중심
- Linear - 미니멀하고 기능적인 UI
- Arc Browser - 모던하고 절제된 컬러
- Notion - 깔끔한 타이포와 여백

**지양하는 스타일**:
- ❌ 카카오톡 노란색 같은 강한 브랜드 컬러
- ❌ 과도한 그라데이션, 네온, 글로우 효과
- ❌ 귀여운 일러스트, 캐릭터, 이모티콘 남발
- ❌ 복잡한 레이어, 카드 중첩, 과한 그림자

### 2. 색상 시스템

**Primary Palette** (메인 UI):
```
현재는 모노크롬 베이스 (#212529 검정, #f8f9fa 회색, 흰색)
→ 제안 필요: 서울 지하철 + 출퇴근 + 모던 감성을 표현하는 Primary Color

예시 방향성:
- Option A: 차분한 블루-그레이 계열 (신뢰감, 도시적)
- Option B: 따뜻한 그레이-베이지 계열 (친근함, 편안함)
- Option C: 서울 지하철 노선 컬러에서 영감 (2호선 그린, 3호선 오렌지 등)
```

**Accent Colors** (익명 사용자 구분용):
```
현재: 16가지 비비드 컬러 사용 중
→ 제안: 톤앤매너에 맞는 파스텔/muted 컬러 팔레트
- 충분히 구분 가능하면서도 시각적으로 조화로운 8-12색
```

**Functional Colors**:
```
- Success (전송 완료):
- Warning (삭제 경고):
- Error (네트워크 오류):
- Info (시스템 안내):
```

### 3. 타이포그래피

**현재 상태**:
```css
제목: 20px / 700
본문: 15px / 400
캡션: 12px / 500
```

**개선 요청**:
- 한글 가독성 최적화 폰트 추천 (웹폰트)
- 서울/지하철 감성에 맞는 폰트 페어링
- 메시지 말풍선 내 최적 폰트 사이즈/행간
- 모바일에서 피로도 낮은 자간/행간 설정

**참고사항**:
- 입력창은 16px 이상 필수 (iOS 자동 줌 방지)
- 지하철 안 흔들림 환경에서도 읽기 편해야 함

### 4. 화면별 디자인 개선 요청

#### A. 메인 화면 (호선 목록)

**현재 상태**:
```
- 상단: "출퇴근하는 노선의 채팅방에 참여하세요" 헤더
- 탭: 호선 순 / 인기 순
- 리스트: 9개 호선 카드 (좌측 컬러바, 활동 상태 표시)
- 하단: "매일 9시 리셋" 힌트
```

**디자인 개선 포인트**:
1. **첫인상 최적화**:
   - 현재 헤더가 밋밋함 → 시선을 끄는 임팩트 필요 (과하지 않게)
   - 서비스 특성(익명, 임시, 지하철) 직관적으로 표현하는 비주얼 요소

2. **호선 카드 디자인**:
   - 현재: 좌측 4px 컬러바 + 텍스트 정보
   - 개선: 각 호선의 정체성을 더 강하게 표현하는 방법?
   - "대화 공간" 느낌을 주는 비주얼 요소
   - 활동 상태 (N명 대화 중)를 더 매력적으로 표현

3. **인터랙션**:
   - 카드 호버/탭 시 피드백 개선
   - 로딩 상태 (3초마다 갱신) 표현 방법

#### B. 채팅방 화면

**현재 상태**:
```
- 헤더: ← [호선명] (56px 고정)
- 메시지: 70% 최대 너비 말풍선
  - 내 메시지: 우측, 검정 배경
  - 상대 메시지: 좌측, 흰 배경 + 좌측 3px 컬러바
- 하단: 입력창 + 전송 버튼
```

**디자인 개선 포인트**:
1. **말풍선 스타일**:
   - 현재 검정(내 메시지)이 너무 무거운가?
   - 좌측 컬러바만으로 익명 구분이 충분한가?
   - 말풍선 모서리, 그림자, 여백 최적화
   - 답장 프리뷰 디자인 개선

2. **Empty State (첫 메시지 전)**:
   - 현재: 💬 이모지 + 안내 텍스트
   - 개선: 첫 대화 시작을 유도하는 디자인

3. **입력 영역**:
   - 현재: 회색(비활성) → 검정(활성) 전송 버튼
   - 개선: 더 직관적이고 매력적인 인터랙션
   - 플레이스홀더 문구 개선 ("메시지를 입력하세요" → ?)

4. **시스템 메시지**:
   - 날짜 구분선 ("오늘", "어제")
   - 시간 표시 (오전/오후)
   - 이 요소들의 시각적 위계 개선

### 5. 컴포넌트/패턴 디자인

**필요한 UI 컴포넌트**:
1. **Button States**: Default / Hover / Active / Disabled / Loading
2. **Input States**: Empty / Focused / Filled / Error
3. **Card States**: Default / Active / Loading
4. **Toast/Alert**: Success / Error / Info
5. **Loading Indicator**: 메시지 갱신 시 (5초마다)
6. **Empty States**: 메시지 없음 / 네트워크 오류

**인터랙션 개선**:
- 스와이프 답장 시 비주얼 피드백 강화
- 새 메시지 도착 애니메이션
- 스크롤 다운 버튼 디자인
- 삭제 확인 모달 디자인
